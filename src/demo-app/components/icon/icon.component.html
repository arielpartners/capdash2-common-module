Icon Demo

Basic

You can simply add the `icon` attribute on the cd-icon selector to display any Font-Awesome icon

Set Icon Size

Add size attribute on cd-icon to enlarge current icon. For more detail, checkout source file of font-awesome larger.less

Set Icon Background Color

Add the 'color' attribute to cd-icon to change background color

Set Icon Fixed Width

Add the 'fixed' attribute to cd-icon and set to 'true'
